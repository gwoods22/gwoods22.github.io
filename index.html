Grade Scraper

I wrote a Puppeteer JS web scraping script that logs into McMaster University's grade portal to pull final grades and store a screenshot in an S3 Bucket. I deployed the script as an AWS Lambda function to run the script every hour and using the Twilio API I sent myself a text with a screenshot from S3 when new grades were posted.
